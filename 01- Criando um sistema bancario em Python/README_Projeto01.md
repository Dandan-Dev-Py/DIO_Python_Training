# Este é um projeto para a plataforma DIO

# Estes projetos, obrigatoriamente não terão utilização de IA ou suporte, Apenas projetos autorais

O projeto 01 consiste em criar uma interface simples de um tipo de banco/caixa eletronico, ele tem as opções de deposito, saque e extrato

deposito - pede um valor por meio de input() e se esse valor for maior que 0, o deposito é realizado com sucesso, caso não seja, o deposito não é completado

saque - a função saque pede o valor a ser sacado e, caso o valor seja menor ou igual ao valor que tem em conta, e, seja menor que o limite de saque, e, caso o limite diario de saques esteja menor que o limite maximo de saques, o saque é realizado. caso qualquer uma dessas circunstancias esteja invalida, o saque não é completado

extrato - retorna o extrato das transações realizadas
